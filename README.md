# 💰 AgentWallet AI

## 🚀 Overview
AgentWallet AI is a **multi-agent autonomous financial decision system** that analyzes user spending in real-time and provides intelligent actions like **restrict, allow, or invest**.

---

## 🧠 Core Idea
Instead of a simple expense tracker, this system uses **multiple AI agents**:

- 💰 Save Agent → Prevents overspending  
- 😎 Spend Agent → Supports lifestyle spending  
- 📈 Invest Agent → Encourages future growth  

These agents **compete and collaborate** to give the best decision.
